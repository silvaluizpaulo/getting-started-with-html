World Wide Web (WWW)

HTML => HyperText Markup Language

File types found on the web:
- plain text
- HTML
- images
- videos
- music

Major pieces
A Person using a Computer/Browser, connecting to the internet through HTTP protocol requests the Server information.

HTTP: main protocol of the web
Servers: computers that host the files that make up the web
Internet: the world's largest computer network
Browser: a program that runs on a computer to display files found on the web

HTML
- heart of the web
- made up of: text content, markup, references to other documents and links to other pages

HTML Markup
- made of tags
